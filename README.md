# Titanic Survival Analysis

## Project Overview
This project analyzes the Titanic dataset using Python and Exploratory Data Analysis (EDA).

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Survival Analysis
- Gender Distribution
- Passenger Class Analysis
- Fare Distribution
- Correlation Heatmap

## Key Findings
- Females had higher survival rates than males.
- First-class passengers survived more frequently.
- Most passengers were between 20 and 40 years old.
- Most tickets belonged to third-class passengers.
